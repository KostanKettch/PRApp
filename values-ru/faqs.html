Как найти подкасты?
Есть два способа найти новые подкасты:
  • Просматривайте лучшие подкасты на вкладке "Лучшие чарты".
  • Поиск подкастов по названию на вкладке "Поиск".

 

Как подписаться на подкаст?
После того, как вы нашли подкаст в списке лучших чартов или в списке результатов поиска подкастов, вы можете смахнуть элемент подкаста в списке влево, чтобы открыть меню подписки.
Или вы можете нажать на название подкаста, чтобы открыть подкаст, а затем нажать кнопку «Подписаться», чтобы подписаться на него.

 

Как отписаться от подкаста?
Чтобы удалить подкаст из подписки, сначала перейдите на вкладку «Подкасты», а затем найдите подкаст. Затем смахните элемент подкаста в списке влево, чтобы открыть кнопку удаления.
Или коснитесь названия подкаста, чтобы открыть его, а затем нажмите Настройки, чтобы открыть индивидуальные настройки подкаста. Затем используйте кнопку «Отписаться», чтобы отказаться от подписки.

 

Можно ли воспроизводить через приложение музыку/загрузки/аудиокниги?
Конечно, вы можете воспроизводить свою музыку/загрузки/аудиокниги с помощью этого приложения. Для этого вам нужно настроить “виртуальный” подкаст. Он будет использовать каталог на локальном устройстве как “виртуальный” репозиторий подкастов. Все медиафайлы в каталоге будут импортированы в базу данных и обработаны как выпуски этого “виртуального” подкаста.

 

Как добавить подкаст вручную?
Если ваш подкаст не найден поиском, вы всегда можете добавить подкаст вручную. Перейдите на вкладку «Подкасты» и нажмите кнопку «Добавить» в строке заголовка. Затем выберите в меню Добавить подкаст по URL. Введите свою ссылку на подкаст, а затем нажмите кнопку «Найти подкаст». Если подкаст будет найден, он будет добавлен в приложение.

 

Что означают синие и зеленые кружки прогресса?
  • Синий круг – это индикатор хода воспроизведения.
  • Зеленый круг — это индикатор загрузки.

 

Что означают синие и зеленые цифры на вкладке "Подкасты"?
  • Синее число — это общее количество невоспроизведенных серий.
  • Зеленое число — это количество последних выпусков, полученных с момента последнего открытия подкаста.
    Этот номер будет сброшен на ноль, как только вы откроете подкаст для просмотра списка выпусков.

  1. Проведите вправо от левого края экрана, чтобы открыть боковое меню навигации
  2. Нажмите Настройки, чтобы открыть экран настроек приложения.
  3. Перейдите к настройкам Подкасты
    4. .
  4. Нажмите Проверить обновления и выберите вариант из списка.

 

Почему мой подкаст не обновляется?
Вы знаете, что публикуются новые выпуски, но приложение не получает эти новые выпуски. Что ж, очень вероятно, что в RSS-канале подкаста есть опечатка/ошибка даты публикации. Приложение использует дату публикации каждого выпуска в качестве ключевого индикатора того, новый ли это выпуск, который нужно извлечь, или старый, который нужно пропустить. Однако иногда дата публикации некоторых выпусков может быть неверной из-за опечатки или по каким-то другим причинам. Если эта неправильная дата публикации на несколько месяцев позже текущей даты, это не позволит приложению обновить подкаст, так как все выпуски до этой неправильной даты будут пропущены как ложные “более старые“ выпуски. В этом случае самый быстрый способ заставить приложение снова обновить подкаст — это сбросить данные выпусков

 

Что такое главы?
Главы или закладки, как это названо в некоторых других приложениях, представляют собой примечания к определенным точкам на временной шкале аудио- или видеофайла. Они особенно полезны для продолжительных выпусков. Вы можете нажать на главу, чтобы начать воспроизведение с этой конкретной позиции на временной шкале.

Некоторые издатели подкастов могут добавлять информацию о главах в медиафайл подкаста. В этом случае это приложение извлечет их и покажет вам. Это приложение также может получать информацию о главах в шоу-нотах подкастов, если они есть. Пользователи также могут добавлять главы вручную.

 

Что означает символ E?
Символ E означает Explicit(Откровенный). Выпуск может быть помечен как откровенный, если он содержит откровенный контент. Откровенный контент может сделать его неприемлемым для детей или потенциально оскорбительным для некоторых слушателей.

Обычно это относится к:
  • Нецензурная брань
  • Ссылки на насилие, которое может быть физическим, психическим или другим насилием.
  • Дискриминация
  • Отсылки к сексуальной активности или поведению.

 

Как получить доступ к индивидуальным настройкам подкастов?
  • Перейдите на вкладку "Подкасты".
  • Нажмите на название подкаста, чтобы открыть его.
  • Нажмите кнопку "Настройки" на экране, чтобы открыть настройки отдельных подкастов.

 

Что такое критерии уникальности выпуска?
Критерии уникальности выпуска — это критерии, используемые для определения уникальности выпуска. Этот критерий используется для предотвращения дублирования выпусков, извлеченных из фида подкастов. Многие издатели подкастов присваивают выпуску глобальный уникальный идентификатор (GUID), чтобы однозначно идентифицировать каждый выпуск. Однако стандарт RSS-канала подкастов не требует этого. Некоторые издатели подкастов вообще не используют GUID. Вместо этого они используют URL-адрес выпуска, дату публикации выпуска или название выпуска в качестве уникального способа идентификации выпуска.

По умолчанию это приложение попытается определить уникальные критерии, используемые в фиде подкастов, и удалить повторяющиеся выпуски из фида подкастов. Он должен работать нормально большую часть времени. Если автоматически определенные критерии уникальности не работают и выпуски дублируются, вы можете перейти к индивидуальным настройкам подкастов, чтобы установить критерии уникальности этого выпуска вручную

 

Что делает сброс выпусков?
Данные выпуска будут локально кэшироваться этим приложением в базе данных приложения, когда они будут получены с сервера подкастов. Сброс выпусков отбрасывает данные выпуска, кэшированные в базе данных, и снова перестраивает их с сервера подкастов. Это полезно, если кэшированные данные выпуска повреждены из-за
различным причинам или имеют ошибки в данных выпуска.

 

Как работает автоматическая загрузка?
Параметр автоматической загрузки можно включить либо на странице настроек приложения, либо настроить для каждого подкаста в индивидуальных настройках подкастов. После включения служба автоматической загрузки будет работать в фоновом режиме и периодически автоматически загружать последние выпуски, которые она находит для вас.

Самые последние выпуски — это новые выпуски с момента последнего просмотра подкаста.

  • Примечание. Если ваше устройство перезагружено, вам нужно сначала запустить приложение, чтобы снова запустить службу автоматической загрузки.

 

Можно ли включать/выключать автоматическую загрузку определенных подкастов?
Да, функцию автоматической загрузки можно настроить для каждого подкаста с помощью индивидуальных настроек подкаста. Индивидуальные настройки подкастов имеют приоритет над системными настройками.

 

Что такое умная загрузка?
Если включена функция Умная загрузка, это приложение будет автоматически загружать до x новых и невоспроизведенных выпусков из текущего воспроизводимого подкаста всякий раз, когда вы начинаете воспроизводить выпуск. Эта функция очень полезна, если у вас большой отставание или если вы любите играть выпуски в хронологическом порядке. Когда вы начнете воспроизводить выпуск с даты X, приложение начнет загрузку более новых выпусков (X+1, X+2...) в фоновом режиме, чтобы вы могли продолжить воспроизведение этих более новых выпусков в автономном режиме, когда закончится текущий выпуск X!

 

Что такое Оставить загруженным?
Для индивидуальных настроек подкаста это максимальное количество загрузок, которое будет сохранено для каждого подкаста. Если в список загрузки добавляется новая серия, превышающая это ограничение, самый старый загруженный файл будет автоматически удален.

 

Как сохранить загрузки на SD-карту?
Перейдите в настройки приложения и укажите место загрузки для использования SD-карты в настройках Загрузки.

 

Почему загрузка приостановлена для Wi-Fi?
По умолчанию это приложение загружается только при подключении к Wi-Fi. Если Wi-Fi недоступен, загрузка будет приостановлена и возобновлена, когда Wi-Fi станет доступен.

 

Почему загрузка приостанавливается из-за низкого заряда аккумулятора, когда он подключен к сети для зарядки?
Система Android отправит предупреждение о батарее LOW, если уровень заряда батареи опустится ниже 15%. Однако система Android не будет отправлять заряд батареи OK до тех пор, пока уровень заряда батареи не превысит 20 %. В результате, если загрузка приостановлена из-за низкого заряда батареи, она не будет возобновлена, даже если устройство подключено к сети, пока система Android не достигнет состояния батареи OK.

 

Можно ли скачивать через подключение для передачи данных 3G или 4G?
Да, отключите Загрузить только через Wi-Fi в настройках приложения.

 

Что означает сообщение об ошибке загрузки?
  • Не удалось найти URL: Загружаемый подкаст недоступен на сервере. Скорее всего, подкаст был удален издателем подкаста или он недоступен при вашем подключении к Интернету.
  • Недостаточно места. Недостаточно свободного места в хранилище.
  • Ошибка: Во время загрузки произошла неизвестная ошибка.

 

Можно ли возобновить неудачную загрузку?
Да, длительное нажатие на элемент, который не удалось загрузить, на экране загрузок вызовет меню. Выберите «Повторить загрузку» в меню, чтобы снова загрузить элемент.

  1. Откройте настройки приложения
  2. Перейдите в настройки Загрузки
    3. .
  3. Включите параметр Удалить загруенное, когда выпуск воспроизведен на 100 %

 

Как сохранить подкаст навсегда?
Если вы хотите сохранить копию подкаста на постоянной основе, сначала загрузите подкаст, а затем экспортируйте загруженный подкаст в папку на вашем устройстве. Экспортированный файл подкаста будет сохранен с использованием названия подкаста в качестве имени файла. Доступ к функции экспорта подкастов можно получить на различных экранах. Нажмите и удерживайте элементы на вкладке Загрузки или в раскрывающемся меню на экране Воспроизведение.

 

Почему у меня повторяются загрузки?
Это приложение использует сложный алгоритм для определения критериев уникальности выпуска, используемых в ленте подкастов, для удаления повторяющихся выпусков. Однако это автоматическое обнаружение иногда может дать сбой, в результате чего повторяющиеся выпуски извлекаются как новые выпуски и загружаются. В случае сбоя автоматического определения критериев уникальности выпуска рекомендуется установить эти критерии в индивидуальных настройках подкастов вручную.

Есть ли способ узнать, был ли скачан выпуск?
После завершения загрузки кнопка загрузки изменится на галочку, указывающую на то, что загрузка завершена.

 

Как переместить каталог загрузки в новое место?
Откройте настройки приложения и перейдите в настройки загрузки. Нажмите на параметр Место загрузки, чтобы выбрать новый каталог. Затем приложение автоматически переместит существующие загрузки из текущего каталога загрузки в выбранный каталог. Нет необходимости перемещать загрузки вручную. Убедитесь, что у вас достаточно места для размещения всех существующих загружаемых файлов в новом месте.

 

Почему мой аудиоподкаст воспроизводится как видео?
На некоторых устройствах формат носителя подкаста может определяться неправильно. В этом случае перейдите на страницу индивидуальные настройки подкастов и установите тип носителя вручную.

 

Можно ли воспроизвести видеоподкаст только как аудио?
Перейдите в раздел индивидуальные настройки подкастов и установите для типа мультимедиа значение Аудио, что заставит приложение воспроизводить видеоподкаст только как аудио.

 

Почему воспроизведение останавливается через определенное количество минут в фоновом режиме?
Обычно это происходит, если в настройках Android включена экономия заряда батареи или оптимизация. Попробуйте отключить экономию заряда батареи или исключить это приложение из оптимизации в настройках Android. Существуют также некоторые сторонние приложения, такие как приложения для управления задачами/памятью или антивирусные приложения, которые могут убивать запущенные приложения в фоновом режиме. Если у вас установлены эти приложения, попробуйте добавить это приложение в белый список, чтобы оно не было уничтожено этими приложениями.

 

Громкость подкаста слишком мала. Могу ли я увеличить громкость звука?
Да, в приложении поддерживается усиление звука. Потяните вверх мини-плеер в нижней части экрана, чтобы развернуть его до полноэкранного, а затем нажмите кнопку «Дополнительно» (значок с тремя точками), чтобы открыть меню воспроизведения. Затем выберите в меню пункт «Аудиоэффекты и эквалайзер». Вы можете включить усиление звука как для одного подкаста, так и для всех подкастов.

 

Есть ли в приложении таймер сна, который останавливает воспроизведение через установленное время?
Да, таймер сна поддерживается в приложении. Потяните вверх мини-плеер в нижней части экрана, чтобы развернуть его до полноэкранного. Таймер отключения находится в строке элементов управления воспроизведением.

Как выполнить трансляцию в Chromecast?
Chromecast поддерживается этим приложением. Значок Chromecast появится в правом верхнем углу экрана Воспроизведение при обнаружении любого Chromecast устройства.

 

Что такое умная перемотка?
Умная перемотка — это функция в этом приложении, которая автоматически перематывает от 5 до 60 секунд в зависимости от того, как долго воспроизведение было приостановлено или остановлено. Чем дольше воспроизведение было приостановлено или остановлено, тем дольше будет применяться время перемотки назад.

 

Как отключить автоматическое воспроизведение Bluetooth?
Некоторые Bluetooth-динамики или автомобильные комплекты автоматически отправляют команду Воспроизвести при подключении к Android. После этого Android запустит последнее воспроизведенное приложение независимо от того, запущено оно или нет. Чтобы остановить это автоматическое воспроизведение в этом приложении, перейдите в Настройки приложения > Дополнительно, чтобы включить параметр Обойти автовоспроизведение Bluetooth.

Если этот параметр Обойти автовоспроизведение Bluetooth включен, это приложение не будет отвечать на команду воспроизведения, если выполняются следующие условия:

  • Приложение работает в фоновом режиме, экран заблокирован или выключен.
  • В данный момент приложение не воспроизводится или находится в состоянии паузы.

 

Почему в Android 11 несколько уведомлений о воспроизведении накапливаются?
В Android 11 появился новый элемент управления мультимедиа, который группирует все мультимедийные уведомления в быстрые настройки. По умолчанию этот новый элемент управления мультимедиа отображается все время, независимо от того, воспроизводится подкаст или нет.

К сожалению, этот новый элемент управления мультимедиа содержал много ошибок при выпуске Android 11. Несколько медиа-уведомлений могут накапливаться на панели уведомлений.

Google исправила эту ошибку в одном из системных исправлений. Однако системное исправление установлено лишь для нескольких устройств, таких как телефоны Google Pixel.

Если производитель вашего устройства еще не предоставил вам это системное исправление, вы можете обойти эту ошибку, перейдя в Настройки приложения > Уведомления и включив Использовать устаревшие мультимедийные уведомления.

 

What are the play modes?
Currently there are seven playback modes supported in this app:
  • Shuffle mode: plays all episodes in the same playlist or episode list but in a random order
  • Priority mode: finish current episode and then switch to play the highest (non-empty) priority playlist
  • Playlist mode: plays all episodes in the same playlist or episode list in serial
  • Repeat playlist mode: play all episodes in the same playlist or episode list repeatedly till it is stopped by user
  • Single episode mode: play one single episode and then stop, keep current episode in Now playing.
  • Single episode (load next): play one single episode and then stop, load next episode into Now playing.
  • Repeat episode: play the same episode repeatedly till it is stopped by user

Default play mode setting is available in Media player settings. Each playlist can have its own play mode which can be setup through playlist setup (Playlists > More > Manage playlists).

Currently selected play mode will be displayed on top of the Up next list in the Playing screen.

How does the Up next list work?
The Up next list serves as the play queue in the app. This play queue is a dynamic queue which will update itself automatically when switching to a new playlist or new episodes are added to the playlist or ordering in the playlist has been changed. For example, you go to the Playlists tab and tap episode in the “playlist A” to start playing. Then, the Up next list will be populated with all episodes from “playlist A”.

Later, you go to the Downloads tab and tap episode in the Downloads list to start playing. Now, the Up next list will be populated with all episodes from the Downloads list. Also, when the episode ordering is changed in the playlist, that change will be synced to the Up next list automatically. You don’t need to do anything manually to update the play queue.

 

How does the playlist work?
In this app any episode list can be played as a playlist. You can click the episode in the episode list to start playing. The episode list will then be populated to the Up next list. There are two types of playlists based on the features supported:
  • Full featured playlist: the playlists created in the Playlists tab
  • Simple playlist: all the other episode lists in the app
A priority value can be assigned to a playlist. When playing with Priority mode, after current episode finishes the player will automatically switch to play the highest (non-empty) priority playlist.

How to transfer downloaded podcast files to another device and get recognized by this app?
Suppose you want to transfer downloaded podcast files from device A to device B, please follow the steps below to get the downloaded podcasts to be recognized by this app on device B:
  1. Start the app on device A
  2. Go to app settings > Backup & Restore > Backup to create a backup of the app data on device A
  3. Copy the app data backup file to device B
  4. Create a new directory on device B
  5. Copy the downloaded podcast files from device A to this newly created directory on device B
  6. Start the app on device B
  7. Follow the Restore app data instructions to restore from the backup of the app data on device B
  8. Go to app settings > Downloads to set up the download directory on device B
  9. Go to the Downloads tab and open the overflow menu
  10. Select Import downloads
  11. Select the directory where you have the podcast files copied from device A.

Wait for the importing to finish. You can delete the copied podcast files once verifying the downloads are recognized again.

 

How to restore app data from backups?
  1. With your Android device powered on, connect it to a computer's USB port.
  2. In Windows Explorer (or the Mac equivalent) select the Android device and then its internal storage.
  3. Open the backup directory. You can find where the backup directory is under the Backup & Restore settings. Copy these files to a location on your PC, as a backup precaution.
  4. Locate the download directory and copy all the files to a location on your PC, as a backup precaution.
  5. Start Podcast Republic, go to app settings > Downloads to set up the download location.
  6. Locate your download directory on your device. If the directory is empty, connect your Android device to the PC again. Copy the downloaded podcast files from your PC to the download directory on your Android device.
  7. Start Podcast Republic, go to app settings > Backup & Restore > Restore.
  8. Locate the backup directory.
  9. Open this directory and locate the most recent backup file. They are named "backup_YYYYMMDDHHMM.zip" based on the date and time when the backup was taken.
  10. Long press the most recent backup file and then tap the Select button. This will recover all of your subscriptions, played/unplayed lists, favorites, etc.
  11. Depending on when the backup was taken, you may be missing some recent activity. You will need to manually add any subscriptions that were added after your most recent backup was taken.
  12. For any episodes that you have played since your most recent backup was taken, you will need to find the episode, tap the "..." next to it and select "Set Played" to mark it as already played.
  13. If you marked any episodes as favorites (using the heart) after the most recent backup was taken, you will need to manually mark those as favorites again from memory.

Why the app doesn't update or stop playing when the app goes to background or is closed?
This is mostly due to the battery optimization features built into the Android system. Please go to your Android Settings to turn off the battery optimization for this app. The exact settings can vary from device to device. Typically you can look for those Settings in the following locations:
  • Android Settings > Apps > Podcast Republic > Battery usage > Battery optimization. Please make sure this option is set to Not optimized.
  • Android Settings > Apps > Podcast Republic > Battery usage > Background restriction. Please make sure this option is set to App can use battery in background.
  • Android Settings > Apps > Podcast Republic > Battery > Allow background activity. Please make sure this option is checked.
  • Android Settings > Device care > Battery > App usage > Put this app to sleep when it's not in use. Please make sure this option is turned off.
Please check out this website https://dontkillmyapp.com/ for more info.

How does this app access the storage?
Android 4.4 introduces the Storage Access Framework (SAF) to secure user’s files. SAF allows users to grant the access permission to a single file or directory dynamically instead of granting the storage permission to the whole storage blankly during the app installation. This provides better privacy protection for users. User privacy is the #1 consideration of this app. Therefore, this app uses SAF when dealing with your storage.

Any time this app needs to access a file or directory it will redirect your to Android's built-in file manager app to complete the select the file or directory. Android system will then grant the access permission to the selected file or directory to this app. If you use a third party file manager app then this procedure may break depends on how well your file manager app supports SAF. We strongly suggest to use Android's built-in file manager app to avoid any issues when dealing with your storage.

Why podcast info isn't displayed via Bluetooth in my car?
Typically that's due to the incompatible AVRCP versions on your phone and your car Bluetooth. To fix that, first, you'll need to have Developer options enabled in your Android Settings. Open your Android Settings and then go to About Phone, then tap the Build Number 7 times. After Developer options is enabled, go into Developer Options, scroll down to Bluetooth AVRCP Version. Change AVRCP version to a lower version, restart the phone. You may need to try couple times to find the right AVRCP version which is compatible with your car.
